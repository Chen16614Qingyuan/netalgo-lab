# SimpleDNS with DPDK

使用DPDK以及[SimpleDNS](https://github.com/mwarning/SimpleDNS)实现了一个示例DNS服务。

本项目是作者在2019年中国科大网络算法学课程中完成的实验内容。

关于具体的实验要求，请查看[scripts](scripts/)目录下文件。

本项目的实现细节请查看[reports](reports/)目录下文件。

## Usage

编译

```bash
make
```

运行

```bash
sudo ./build/server-main
```
